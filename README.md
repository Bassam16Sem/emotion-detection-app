Pendeteksi Emosi (Emotion Detection System)
Deskripsi Proyek
Proyek ini adalah aplikasi berbasis Python yang menggunakan Watson NLP Library untuk menganalisis emosi dari sebuah teks. Aplikasi ini dapat mengidentifikasi berbagai tingkat emosi seperti anger (marah), disgust (muak), fear (takut), joy (senang), dan sadness (sedih), serta menentukan emosi mana yang paling dominan.

Aplikasi ini dideploy menggunakan framework Flask untuk menyediakan antarmuka web yang memungkinkan pengguna memasukkan teks dan melihat hasil analisis secara langsung.

Fitur Utama
Menganalisis teks input untuk mendeteksi emosi.
Memberikan skor numerik untuk setiap kategori emosi.
Menentukan Dominant Emotion dari teks yang diberikan.
Menangani input kosong dengan pesan kesalahan yang sesuai.
Dilengkapi dengan Unit Testing untuk memastikan akurasi fungsi.
Teknologi yang Digunakan
Python 3.x
Flask (Web Framework)
Watson NLP API
Requests (Library HTTP)
Pylint (Static Code Analysis)
